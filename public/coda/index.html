coda

チームの access と knowledge を AI agent に共有する。agent はツールを操作し、学んだことを skill に書き戻す。

Codatum, Inc. の社内 AI skills。ローカルの AI エージェントから直接使う。Slack / GitHub の bot としても動く。

01

Access

チームの credential を agent に共有する。agent はチームと同じツールにアクセスできる。個人の権限を委譲するのではなく、組織の access を共有する。

02

Knowledge

その access をどう使うかを skill に書く。マークダウンに構造化された手順。AI が読んで、そのまま実行する。

03

Growth

agent はタスク完了後に skill を PR で更新する。workspace はほぼ開放されており、使うほど知識が育つ。

最後のステップが重要。agent はタスクを実行するだけでなく、手順に曖昧さや不足があれば skill 自体を修正する。この循環により、チームの知識ベースが使うほど洗練されていく。
